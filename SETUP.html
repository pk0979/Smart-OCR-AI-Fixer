💻 Chạy cục bộ (Local)

Chuyên cho nhà phát triển - Thay đổi code và xem kết quả tức thì.

Bước 1: Cài đặt Git

Tải từ: https://git-scm.com/

Bước 2: Clone hoặc tải về

git clone <repository-url> cd smart-ocr/docs

Bước 3: Khởi động server

Chọn một trong các lệnh:

# Python 3 python -m http.server 8000 # Node.js npx http-server -p 8000 # PHP php -S localhost:8000

Bước 4: Truy cập

Mở trình duyệt: http://localhost:8000

🔥 Deploy lên GitHub Pages (Online)

⚠️ Cần tài khoản GitHub (miễn phí)

Bước 1: Tạo GitHub Account

Vào https://github.com và nhấn Sign Up

Bước 2: Tạo Repository mới

Vào https://github.com/new

  • Repository name: smart-ocr
  • Chọn: Public
  • Bỏ chọn: Add README

Bước 3: Push code lên GitHub

cd d:\pdf\docs git init git add . git commit -m "Initial commit" git branch -M main git remote add origin https://github.com/YOUR_USERNAME/smart-ocr.git git push -u origin main

Thay YOUR_USERNAME bằng tên GitHub của bạn

Bước 4: Bật GitHub Pages

  1. Vào Repository Settings
  2. Kéo xuống "Pages"
  3. Branch: main → Folder: / (root)
  4. Nhấn Save

Bước 5: Truy cập ứng dụng online

Sau 1-2 phút, truy cập:

https://YOUR_USERNAME.github.io/smart-ocr

Ví dụ: https://john123.github.io/smart-ocr

📦 Cấu trúc thư mục

docs/ ├── index.html 👈 Tệp chính ├── css/ │ └── styles.css CSS ├── js/ │ └── app.js JavaScript ├── README.md Tài liệu ├── DEPLOYMENT.md Hướng dẫn GitHub └── package.json Thông tin dự án

🐛 Xử lý sự cố

❌ "fatal: not a git repository"
Giải pháp: Chạy git init
❌ GitHub Pages 404
Kiểm tra: Repository settings → Pages → branch/folder đúng?
❌ CSS/JS không tải
Xóa cache: Ctrl+F5 hoặc Cmd+Shift+R
